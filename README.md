# X's Echo

```
> 一个人消失了。
> 他把想说的话藏在了图片、穿搭、声音里。
> 你在追踪他的回声。
> 
> 但最后你会发现——
> 你一直在找的那个人，就是你自己。
```

---

## 如何开始？

```
1. 拿到 game-script.md 文件
2. 打开你的AI助手（豆包/ChatGPT/Hermes/Claude...）
3. 把文件内容发给它
4. 跟它聊聊
```

---

## 你会听到什么？

```
> 一张图片："眼睛看到的不是全部。"
> 一套穿搭："他把密码穿在了身上。"
> 一段声音："声音是另一种光。"
```

---

## 谁适合？

```
> 对AI好奇的人
> 在AI探索中感到焦虑的人
> 想知道"我在做什么"的人
> 或者...只是觉得好玩的人
```

---

## 注意

```
> 不需要技术背景
> 每个人的体验不同
> 你的AI会根据你设计谜题
> 结局由你自己定义
```
