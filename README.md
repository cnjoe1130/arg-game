# 钢穴·回声 (Caves of Steel: Echo)

> 基于阿西莫夫《钢穴》改编的AI互动解谜游戏

## 这是什么

一个在浏览器里运行的侦探解谜游戏。你扮演调查员，追踪失踪的数字考古学家X。你的AI助手帮你分析证据，但X也在通过助手向你传递消息——是求助还是挑衅？你需要在真假线索中做出判断。

## 怎么玩

1. 打开 `index.html`（经典Mac风格）或 `index-v2.html`（现代macOS风格）
2. 在终端输入 `connect`，连接你的AI（OpenAI/DeepSeek/本地模型）
3. 跟着助手的指引，使用三个工具破解X留下的加密碎片
4. 判断线索真假，找到X

## 三个工具

- **REDSecret** — 图片隐写分析，提取隐藏信息
- **MelodyDress** — 旋律编码分析，解码穿搭中的密码
- **ToneCanvas** — 频谱分析，发现音频中的隐藏信号

## 技术

- 纯前端HTML，无需服务器
- 通过浏览器直接调用AI API（需自己的Key）
- 也支持本地模型（Ollama等）

## 灵感

致敬艾萨克·阿西莫夫《钢穴》——人类侦探Elijah Baley与机器人搭档Daneel Olivaw的破案故事。
